## Build app

1. Clone repository.
```shell
$ git clone https://github.com/AlexBesida/SlikkerTasker.git
```
2. Navigate to cloned folder, and install packages.
```shell
$ flutter pub get
```
3. Run the app.
```shell
$ flutter run
```
